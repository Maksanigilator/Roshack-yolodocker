Duck Detection Dataset Sources
==============================

All datasets merged into a single unified dataset with one class: "duck" (id=0).
Images without duck annotations are excluded.

1. Real World DuckieTown Duckies (Roboflow)
   URL: https://universe.roboflow.com/duckies-1cfdk/real-world-duckietown-duckies
   Original classes: Duckies, null
   Kept: Duckies -> duck
   ~2001 images, real-world duckietown environment

2. Aerial Only Duck View (Roboflow)
   URL: https://universe.roboflow.com/ducks-pm7jq/aerial-only-duck-view
   Original classes: multiple duck types + null
   Kept: all except null -> duck
   Aerial/top-down view of ducks

3. Rubber Ducks (Roboflow)
   URL: https://universe.roboflow.com/patos/rubber-ducks
   Original classes: rubber_duck, null
   Kept: rubber_duck -> duck
   Various rubber duck images

4. DuckieTown HOSv3 (Roboflow)
   URL: https://universe.roboflow.com/duckies/duckietown-hosv3
   Original classes: Duckie, Duckiebot, etc.
   Kept: Duckie -> duck (removed Duckiebot and other classes)
   Duckietown simulation/real environment

5. Rubber Duck Detection Dataset (HuggingFace)
   URL: https://huggingface.co/datasets/brainwavecollective/yolo-rubber-ducks
   Original classes: rubber_duck (class 0)
   Kept: rubber_duck -> duck
   192 images, rubber ducks on various backgrounds

6. Rubber Duckies Object Detection (Edge Impulse)
   URL: https://studio.edgeimpulse.com/public/783690/latest/acquisition/training
   Original classes: rubber-ducky
   Note: requires manual export from Edge Impulse platform (YOLO TXT format)
   206 images
